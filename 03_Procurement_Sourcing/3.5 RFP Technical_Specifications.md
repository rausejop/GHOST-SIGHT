# 3.5 RFP Technical_Specifications: Pliego de Prescripciones Técnicas (PPT)

## 1. Objeto del Suministro
Suministro, instalación y mantenimiento de una plataforma de monitorización 3D volumétrica para adiestramiento avanzado de unidades MOE, GAR, GEO y FGNE.

## 2. Requerimientos de Hardware (Lote 1)
- **Sensores LiDAR:** Mínimo 128 canales, 20Hz, alcance 50m, precisión < 1cm. Certificación MIL-STD-810H.
- **Nodos Edge:** Capacidad de proceso > 200 TOPS (NVIDIA AGX Orin). Chasis IP67.

## 3. Requerimientos de Software y Algoritmia (Lote 2)
- **Pose Estimation:** Reconocimiento de siluetas humanas 3D en 0 lux y humo denso.
- **Muzzle Discipline:** Alerta en <100ms ante detección de cañón cruzado entre operadores.
- **Interoperabilidad AR:** Exportación de Digital Twin compatible con visores HoloLens 2 / IVAS.

## 4. Requerimientos de Comunicaciones y Seguridad
- **Red:** Aislamiento total (Air-gapped tactical fiber).
- **Cumplimiento ENS:** Nivel Alto. Cifrado AES-256 GCM.
- **Estándar:** Federated Mission Networking (FMN) Spiral 4.
